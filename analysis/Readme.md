# Analysis Files

This folder contains analysis files used for the **Azure Stay Revenue Analysis** project.

## Files Included

### 📊 EDA_hotel_revenue_analysis.xlsx
Excel workbook used for Exploratory Data Analysis (EDA), including:

- Pivot Tables
- Revenue Analysis
- Booking Trend Analysis
- Rate Code Performance
- Customer Segment Comparison
- Chart Preparation for Presentation

## Purpose

These files were used to explore the dataset, identify business insights, and support dashboard/report creation.
